Google Summer of Code Introductory Seminar




27th February,2017




seminar to provide insights about GSoC




Event Overview

The number of selections from IIT Kharagpur had declined significantly after the year 2012. It was about time that the Google Summer of Code program should have been focused and info seminars should have been organized. On 27th February, a number of ex-GSoCers came up and talked about the methodologies of GSoC. There was an enriched Question-Answer session where lots of basic doubts were cleared. A follow-up session was organized after Google announced the organizations list of 2017 where seniors enlightened students on which orgs to select and how to approach them.

As a result, the number of selections from KGP is 29 in 2017.
