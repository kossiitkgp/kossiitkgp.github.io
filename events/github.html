GITHUB WORKSHOP

About the Workshop

The first step towards open source is learning the workflow of Github and knowing the appropriate Git commands. KOSS organises the Github workshop to help the first and second years have an in-depth knowledge of Git so that they can start contributing to the open source world efficiently. In the workshop, the coders have a hands-on experience using Github as they learn from the basics to the comprehensive part of Git.
